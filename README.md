# [Pabellon criollo]( https://darianbetancourt.github.io/pabellonCriollo/)
E un exercicio para practicar o maquetado e tags basicas fazendo uso de ***_HTML5_*** e ***_CSS_***
- listas ordenadas ***ol***
- listas desordenadas ***ul***
- ancoras ***a***
- titulos, sub titulos ***h1*** ***h2***
- imagenes ***img***
- videos
